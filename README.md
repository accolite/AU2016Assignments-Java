# Office Adv Posting

* An internal Accolite app where all employee can sell or buy items or
post and adv among themselves.

* All employees should able to post on the application for selling any
item. With details of contact info, photos, desc, etc.

* For those who posted an item, should able to edit the status and should
have provision to delete the posted item as well

* It should have a search functionality (Basic search with filters like
product name,category,price, location, etc)

* should able to connect with any employee who have uploaded an item to
sell.

* Ability to see Transaction history for each user.

* Wall page with recent advertisements. [Home page.]

* Google Auth sign-on.

**Admin**: 
* See list of posts in each category, location, price range.
* Can create categories. 
* Can delete ad if inappropriate or blacklist emp.
